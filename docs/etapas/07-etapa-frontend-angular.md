GradeFlow - Etapa 07 - Frontend Angular

Objetivo
Criar uma interface simples para usar o MVP.

Tecnologias
- Angular
- TypeScript
- Reactive Forms
- HttpClient
- Angular Router

Telas do MVP
- Lista de avaliacoes
- Criar avaliacao
- Detalhe da avaliacao
- Criar questao
- Cadastrar gabarito
- Criar submissao manual
- Ver resultado da correcao

Estrutura sugerida
src/app/
  core/
  shared/
  features/
    assignments/
    questions/
    submissions/
    correction/

Services principais
- AssignmentApiService
- QuestionApiService
- SubmissionApiService
- CorrectionApiService

Tarefas
- Criar projeto Angular
- Configurar rotas
- Criar componentes iniciais
- Criar services para chamar API
- Criar formularios reativos
- Integrar tela de avaliacao com API
- Integrar submissao manual com API
- Integrar botao de correcao

Ponto de atencao
Nao gaste tempo com dashboard bonito agora.
Primeiro faca o usuario criar, responder e corrigir uma avaliacao.
