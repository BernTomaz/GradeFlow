GradeFlow - Etapa 07 - Frontend Angular

Objetivo
Criar uma interface simples para usar o MVP.

Tecnologias
- Angular
- TypeScript
- Reactive Forms
- HttpClient
- Angular Router

Telas do MVP
- Lista de avaliações
- Criar avaliação
- Detalhe da avaliação
- Criar questão
- Cadastrar gabarito
- Criar submissão manual
- Ver resultado da correção

Estrutura sugerida
src/app/
  core/
  shared/
  features/
    assignments/
    questions/
    submissions/
    correction/

Services principais
- AssignmentApiService
- QuestionApiService
- SubmissionApiService
- CorrectionApiService

Tarefas
- Criar projeto Angular
- Configurar rotas
- Criar componentes iniciais
- Criar services para chamar API
- Criar formularios reativos
- Integrar tela de avaliação com API
- Integrar submissão manual com API
- Integrar botao de correção

Ponto de atenção
Não gaste tempo com dashboard bonito agora.
Primeiro faça o usuário criar, responder e corrigir uma avaliação.
