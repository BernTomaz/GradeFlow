# Banco de Dados

## Visao Geral

O GradeFlow usa SQL Server com Entity Framework Core Migrations.

## Tabelas Principais

- Assignments
- Questions
- AnswerKeys
- Submissions
- StudentAnswers
- CorrectionResults
- CorrectionLogs
- Users

## Migrations

As migrations versionadas criam a estrutura relacional, relacionamentos, indices e evolucoes do modelo.
