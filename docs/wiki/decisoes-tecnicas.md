# Decisoes Tecnicas

Cada decisao tecnica deve registrar:

- Contexto
- Alternativas consideradas
- Motivo da decisao
- Beneficios obtidos

## Por que Angular?

Angular foi escolhido por oferecer uma estrutura completa para aplicacoes web com rotas, formularios reativos, HTTP e organizacao por features.

## Por que ASP.NET Core?

ASP.NET Core foi escolhido por ser uma plataforma moderna para APIs, com boa integracao com C#, Entity Framework Core, autenticacao e testes.

## Por que SQL Server?

SQL Server foi escolhido por combinar bem com Entity Framework Core e com o ecossistema .NET.

## Por que JWT?

JWT permite autenticar requisicoes HTTP de forma stateless e integrar frontend e backend com simplicidade.

## Por que Strategy Pattern?

O Strategy Pattern separa a regra de correcao de cada tipo de questao e evita concentrar logica de negocio nos controllers.

## Por que Entity Framework Core?

Entity Framework Core reduz codigo de acesso a dados, permite migrations versionadas e mantem o modelo relacional integrado ao dominio.

## Por que arquitetura em camadas?

A arquitetura em camadas separa responsabilidades e facilita manutencao, testes e evolucao do projeto.
