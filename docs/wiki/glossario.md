# Glossario

## DTO

Objeto usado para transportar dados entre camadas ou pela API.

## JWT

Token usado para autenticar requisicoes HTTP.

## Middleware

Componente executado no pipeline de requisicoes da API.

## Dependency Injection

Padrao usado para fornecer dependencias a classes sem cria-las diretamente dentro delas.

## Repository Pattern

Padrao usado para encapsular acesso a dados.

## Strategy Pattern

Padrao usado para trocar algoritmos conforme o contexto, como o tipo da questao na correcao automatica.

## CI/CD

Processo automatizado de build, testes, publicacao e deploy.

## Docker

Ferramenta para empacotar aplicacoes e dependencias em containers.

## Entity Framework Core

ORM usado para mapear entidades C# para tabelas do banco de dados.
