# GradeFlow Wiki

## Visao Geral

GradeFlow e um sistema web para criacao de avaliacoes, definicao de gabaritos estruturados, correcao automatica de respostas e revisao manual de resultados.

## Objetivos

- Demonstrar conhecimentos em .NET e Angular.
- Aplicar arquitetura em camadas.
- Automatizar a correcao de respostas estruturadas.
- Permitir revisao manual e auditoria de resultados.
- Servir como projeto de portfolio.

## Tecnologias

### Backend

- .NET 10
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- JWT Authentication

### Frontend

- Angular 20
- TypeScript
- Reactive Forms
- Angular Router
- HTTP Interceptors

### DevOps

- GitHub Actions
- Docker
- Docker Compose
- Deploy em nuvem
